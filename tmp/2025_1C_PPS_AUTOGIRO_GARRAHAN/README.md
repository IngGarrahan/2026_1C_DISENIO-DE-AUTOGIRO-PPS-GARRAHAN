# Diseño Dinámico y Aerodinámico de un Rotor de Autogiro

Proyecto PPS: herramienta numérica para análisis aerodinámico y dinámico de un rotor de autogiro.

## Estructura
- notebooks: simulación completa
- src: módulos del modelo
- data: perfiles aerodinámicos y Reynolds
- docs: documentación
- results: resultados

## Ejecución
```bash
pip install -r requirements.txt
jupyter notebook notebooks/PPS_ALAN_Rev14.ipynb
```
